select box

    - 옵션 설명 -
    ng-model : select box에서 보여질 데이터 list
    selectedValue :  select box에서 선택된값이 담길 scope명
    actionsBox : 전체선택, 전체 해제를 선택할 수 있는 버튼 노출 유무 예) true - 노출
    multiple : 해당 옵션을 주입하면 multi select 기능을 사용할 수 있다.
    dataSize : option 창에서 화면에 노출될 data 개수
    dataLiveSearch : filter 기능 사용 유무 선택 예) true - 노출
    disabled : 해당 옵션 주입 시 특정 scope 값을 이용하여 사용/미사용을 선택할 수 있다. 예) true - 사용

    * 아이콘 사용 시 array 형식 *
        - 아이콘 사용시에는 "icon" key에 노출시킬 아이콘 class값을 주입한다.
        vm.arrayCollection = [
            {name: 'Alice', class: 'Class A', icon: 'glyphicon-euro'},
            {name: 'Bob', class: 'Class B', icon: 'glyphicon-usd'}
        ];
