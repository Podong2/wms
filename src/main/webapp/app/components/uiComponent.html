side nav

  • 사용자는 숨어있는 side menu를 특정버튼을 눌러 노출시킬 수 있다.
  • 노출된 side menu는 content영역을 밀어서 노출이 아닌 content위로 노출 되어야한다.
  • 노출된 메뉴는 메뉴 이외의 영역을 누르거나 메뉴명을 누를 시 화면에서 사라진다.
  • 메뉴는 트리형태의 구로조 표현 가능해야한다.
  • 선택된 메뉴는 active 효과를 적용해야한다.
  • 사이드 바를 고정도 가능해야한다.
  • jhipster처럼 메뉴 오픈 시 content영역이 밀리거나 덮히는 옵션이 있어야한다.

multi select box

  • 단일 선택 및 다중선택이 가능해야한다.
  • 리스트들을 모두 선택과 선택 해제 기능이 제공되어야한다.
  • 선택된 요소들은 select 박스에 count 또는 선택된 요소들의 텍스트가 표시되어야한다.
  • 선택된 요소들의 해제를 option박스의 선택해제 기능과 선택된 요소를 직접클릭하여 해제하는 기능 그리고 select박스에서 해제하는기능이 제공되어야한다.
  • 요소들의 표현은 text 표현 및 아이콘으로도 표현이 가능해야 한다

alerts

  • alert은 오른쪽 상단, toast는 오른쪽하단 표시, 유효성 체크는 body영역
Success! Indicates a successful or positive action.
Info! Indicates a neutral informative change or action.
Warning! Indicates a warning that might need attention.
Danger! Indicates a dangerous or potentially negative action.

Toast

  • 시스템의 메시지 노출시에는 toast로 노출한다.
  • 노출 후 특정 시간이 지난 후 자동으로 닫힌다.
  • 오류 메시지는 자동닫힘이 없고 x 버튼을 눌러야 닫힌다.
  • 메시지별 각각의 컬러 테마가 적용된다.

  • 버튼 혹은 특정요소를 마우스 오버 시 툴팁이 노출된다.
  • 마우스가 요소 영역에서 벗어날 시 툴팁이 사라진다.
  • 표현되는 text는 개행되지 않아야한다.

datePicker

  • 좌우 화살표 또는 월을 선택하여 원하는 달의 달력을 확인할 수 있다.
  • 달력은 자동으로 오늘날짜를 표시해야한다.
  • 월별로 날짜를 출력하고 해당 날짜를 클릭하여 달력값을 얻을 수 있어야한다.
  • 다국어 가능해야한다.
  • input에서 바로 입력이 가능해야한다.
  • 구간검색이 가능해야한다
  • 단순 입력이 가능해야한다.
  • 년단위로 달력이동가능
  • 특정기간만 선택 가능하도록 가능
  • 구간검색이 가능해야한다
  • 시작일과 종료일에 대한 validation check를 하여 메세지 노출 혹은 자동 기입
  • 시작일자는 종료일자보다 작아야한다.
  • 시작일자는 종료일자와 같을수있다.
  • 특정기간만 선택가능하다.
